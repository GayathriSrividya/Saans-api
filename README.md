# Saans API

A Node.js Express API for OpenTelemetry generation.

## Prerequisites

- Node.js (14+)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Gayathrisrividya/Saans-api.git
cd Saans-api
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env` file in the root directory:
```bash
PORT=3000
NODE_ENV=development
```

## Running the API

### Development Mode
```bash
npm run dev
```
### Production Mode
```bash
npm start
```
